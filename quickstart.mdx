# Quick Start Guide

Get your PayRequest integration up and running in just a few steps. This guide will walk you through creating your first payment request and integrating with our API.

## Step 1: Create Your Account

<Steps>
  <Step title="Sign up for PayRequest">
    Visit [dashboard.payrequest.io](https://dashboard.payrequest.io) and create your free account. You'll get instant access to our sandbox environment for testing.
  </Step>
  <Step title="Get your API keys">
    Navigate to **Settings > API Keys** in your dashboard to find your test and live API keys.
    
    <Warning>
      Keep your API keys secure and never expose them in client-side code.
    </Warning>
  </Step>
  <Step title="Choose your integration method">
    Select how you want to integrate PayRequest into your application:
    - **Payment Links** - No coding required
    - **API Integration** - Full programmatic control
    - **Plugins** - Ready-made solutions for popular platforms
  </Step>
</Steps>

## Step 2: Create Your First Payment

<Tabs>
  <Tab title="Payment Links (No Code)">
    The fastest way to start accepting payments:

    1. Go to **Payments > Create Payment** in your dashboard
    2. Fill in the payment details (amount, description, recipient email)
    3. Click **Create Payment Link**
    4. Share the generated link with your customer

    ```
    https://payrequest.io/p/your-payment-id
    ```

    Your customer can pay instantly using this link, and you'll receive real-time notifications.
  </Tab>

  <Tab title="API Integration">
    For developers who want programmatic control:

    <CodeGroup>
    ```javascript Node.js
    const payrequest = require('@payrequest/api');

    // Initialize with your API key
    const pr = new payrequest('your-api-key');

    // Create a payment request
    const payment = await pr.payments.create({
      amount: 2500, // Amount in cents ($25.00)
      currency: 'USD',
      description: 'Product purchase',
      recipient: {
        email: 'customer@example.com',
        name: 'John Doe'
      },
      callback_url: 'https://yoursite.com/webhook'
    });

    console.log(payment.payment_url);
    ```

    ```python Python
    import payrequest

    # Initialize with your API key
    pr = payrequest.PayRequest('your-api-key')

    # Create a payment request
    payment = pr.payments.create(
        amount=2500,  # Amount in cents ($25.00)
        currency='USD',
        description='Product purchase',
        recipient={
            'email': 'customer@example.com',
            'name': 'John Doe'
        },
        callback_url='https://yoursite.com/webhook'
    )

    print(payment['payment_url'])
    ```

    ```php PHP
    <?php
    require_once 'vendor/autoload.php';

    // Initialize with your API key
    $payrequest = new \PayRequest\PayRequest('your-api-key');

    // Create a payment request
    $payment = $payrequest->payments->create([
        'amount' => 2500, // Amount in cents ($25.00)
        'currency' => 'USD',
        'description' => 'Product purchase',
        'recipient' => [
            'email' => 'customer@example.com',
            'name' => 'John Doe'
        ],
        'callback_url' => 'https://yoursite.com/webhook'
    ]);

    echo $payment['payment_url'];
    ?>
    ```
    </CodeGroup>
  </Tab>

  <Tab title="WordPress Plugin">
    For WordPress sites, install our official plugin:

    1. Download the PayRequest plugin from the WordPress repository
    2. Upload and activate the plugin
    3. Go to **Settings > PayRequest** and enter your API key
    4. Use the `[payrequest]` shortcode to create payment buttons

    ```html
    [payrequest amount="25.00" description="Product Name" button_text="Buy Now"]
    ```
  </Tab>
</Tabs>

## Step 3: Handle Payment Notifications

Set up webhooks to receive real-time payment updates:

<CodeGroup>
```javascript Express.js
app.post('/webhook', express.raw({type: 'application/json'}), (req, res) => {
  const event = req.body;
  
  switch (event.type) {
    case 'payment.completed':
      console.log('Payment completed:', event.data);
      // Update your database, send confirmation email, etc.
      break;
    case 'payment.failed':
      console.log('Payment failed:', event.data);
      // Handle failed payment
      break;
    default:
      console.log('Unhandled event type:', event.type);
  }
  
  res.status(200).send('OK');
});
```

```python Flask
from flask import Flask, request

@app.route('/webhook', methods=['POST'])
def handle_webhook():
    event = request.get_json()
    
    if event['type'] == 'payment.completed':
        print(f"Payment completed: {event['data']}")
        # Update your database, send confirmation email, etc.
    elif event['type'] == 'payment.failed':
        print(f"Payment failed: {event['data']}")
        # Handle failed payment
    
    return 'OK', 200
```
</CodeGroup>

## Step 4: Go Live

<Steps>
  <Step title="Test thoroughly">
    Use our sandbox environment to test all payment flows and webhook handling.
  </Step>
  <Step title="Switch to live keys">
    Replace your test API key with your live API key in your application.
  </Step>
  <Step title="Configure webhooks">
    Update your webhook endpoints to point to your production environment.
  </Step>
  <Step title="Start accepting payments">
    Your PayRequest integration is now live and ready to accept real payments!
  </Step>
</Steps>
