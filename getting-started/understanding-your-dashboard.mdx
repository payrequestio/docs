## 🔍 Search and Navigation

### Global Search

The dashboard search bar is your gateway to finding anything quickly:

**Search Capabilities:**
- **Customers**: Find by name, email, or company
- **Orders**: Search by order number or customer
- **Invoices**: Locate by invoice number or amount
- **Products**: Find by name or SKU
- **Transactions**: Search by payment reference

**Search Tips:**
- Use partial matches for broader results
- Combine multiple terms for specific searches
- Use quotation marks for exact phrase matching
- Filter results by type using dropdown options

### Breadcrumb Navigation

Always know where you are with clear breadcrumb trails:
- **Dashboard** → **Customers** → **Customer Profile**
- **Dashboard** → **Products** → **Edit Product**
- **Dashboard** → **Settings** → **Payment Providers**

## 🚀 Pro Tips for Dashboard Mastery

<Accordion>
  <AccordionItem title="💡 Keyboard Shortcuts">
    **Global Shortcuts:**
    - `Ctrl/Cmd + K`: Open global search
    - `Ctrl/Cmd + /`: View all keyboard shortcuts
    - `G + D`: Go to Dashboard
    - `G + C`: Go to Customers
    - `G + P`: Go to Products
    - `G + S`: Go to Settings

    **Navigation Shortcuts:**
    - `Tab`: Move between interactive elements
    - `Enter`: Activate buttons and links
    - `Esc`: Close modals and dropdowns
  </AccordionItem>

  <AccordionItem title="🎯 Workflow Optimization">
    **Daily Routine:**
    1. Check dashboard metrics and alerts
