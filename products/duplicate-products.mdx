# Duplicate Products

The product duplication feature allows you to quickly create copies of your existing products, saving you time when you need to add similar items with slight variations in name, price, or other details.

## Why Use Product Duplication?

Product duplication is perfect when you need to:
- Create variations of existing products (different sizes, colors, or versions)
- Set up similar service packages with different pricing
- Add seasonal or promotional versions of products
- Create products with similar configurations but different details

## How to Duplicate a Product

### Step 1: Navigate to Your Product
1. Go to your dashboard
2. Click on "Products" in the sidebar
3. Find the product you want to duplicate
4. Click on the product to open its detail page

### Step 2: Start the Duplication Process
1. On the product detail page, look for the "Duplicate" button
2. Click the "Duplicate" button
3. A modal window will open with duplication options

### Step 3: Customize Your Duplicate
In the duplication modal, you can:

**Product Name**
- The original name is pre-filled with "(Copy)" added
- Edit the name to whatever you prefer
- Example: "Premium Plan" becomes "Premium Plan (Copy)"

**Product Price**
- The original price is pre-filled
- Change the price if needed for your variation
- Keep the same price if it's identical

**Other Settings**
- All other product settings (description, features, etc.) are copied exactly
- You can edit these after the product is created

### Step 4: Create the Duplicate
1. Review your changes
2. Click "Duplicate Product"
3. The new product will be created and you'll be redirected to it
4. Edit any additional details as needed

## What Gets Copied?

When you duplicate a product, **everything** is copied including:

### Basic Information
- Product description
- Category assignment
- Product type (physical, digital, service, subscription)
- Status (active/inactive)

### Pricing & Billing
- Tax rate settings
- Currency settings
- Subscription settings (if applicable)
- Setup fees and trial settings

### Advanced Features
- Custom fields configuration
- Stock quantity settings
- Image (if any)
- All metadata and technical settings

### What You Can Immediately Change
- **Product name** - Customize in the duplication modal
- **Price** - Adjust pricing in the duplication modal
- **Everything else** - Edit after creation

## Common Use Cases

### Creating Product Variations
**Example: T-Shirt Shop**
1. Create "Basic T-Shirt - Medium"
2. Duplicate it to create "Basic T-Shirt - Large"
3. Change only the name in the duplication modal
4. Repeat for other sizes

### Service Package Tiers
**Example: Consulting Services**
1. Create "Basic Consultation Package"
2. Duplicate to create "Premium Consultation Package"
3. Update name and price in duplication modal
4. Edit the full product later to add premium features

### Seasonal Products
**Example: Holiday Specials**
1. Have "Regular Website Design"
2. Duplicate to create "Holiday Special - Website Design"
3. Adjust price for holiday discount
4. Update description later to mention holiday terms

### Subscription Plan Variations
**Example: Software Plans**
1. Create "Monthly Plan"
2. Duplicate to create "Annual Plan"
3. Change name and adjust pricing
4. Later edit billing frequency settings

## Tips for Effective Product Duplication

### Naming Strategies
- Add descriptive suffixes: "(Large)", "(Premium)", "(Annual)"
- Use clear differentiators: "Basic vs Pro vs Enterprise"
- Include key differences in the name: "With Setup", "No Setup"

### Pricing Considerations
- Calculate variations carefully (discounts, premiums)
- Consider bulk pricing for quantity differences
- Factor in different cost structures for variations

### Post-Duplication Editing
After duplicating, remember to:
1. **Update the description** to reflect differences
2. **Change the image** if the product looks different
3. **Adjust custom fields** if they're different for this variation
4. **Review category assignment** - might belong in a different category

### Organization Tips
- Use consistent naming patterns across similar products
- Group related products in the same category
- Consider using tags to identify product families

## Advanced Workflows

### Bulk Product Creation
For creating many similar products:
1. Start with one well-configured "template" product
2. Duplicate it multiple times
3. Batch edit the copies with your variations
4. This is faster than creating each from scratch

### Testing New Products
Before launching a new product type:
1. Duplicate an existing successful product
2. Modify it to test new pricing or features
3. Launch as a test without affecting the original
4. Keep or remove based on performance

### Seasonal Campaigns
For holiday or promotional periods:
1. Duplicate your regular products
2. Add seasonal naming and pricing
3. Set them as inactive until the campaign starts
4. Activate when ready, deactivate when done

## Troubleshooting

**Q: I duplicated a product but don't see it in my list**
A: Check if the duplicate was created as "inactive". You may need to activate it in the product settings.

**Q: Can I duplicate a product to a different category?**
A: The duplicate is created in the same category as the original. You can change the category by editing the duplicated product.

**Q: What happens to custom fields when I duplicate?**
A: All custom field configurations are copied exactly. You can modify them in the duplicated product if needed.

**Q: Can I duplicate subscription products?**
A: Yes! All subscription settings are copied, including billing frequency, trial periods, and setup fees.

**Q: Is there a limit to how many times I can duplicate a product?**
A: No, you can duplicate products as many times as needed.

**Q: Can I undo a duplication?**
A: You can delete the duplicated product if you don't need it. The original product is never affected by duplication.
