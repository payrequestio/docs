# PayRequest API Reference

The PayRequest API allows you to programmatically create payment requests, manage transactions, and integrate payment functionality into your applications. Our REST API is designed to be simple, reliable, and well-documented.

## Base URL

<CodeGroup>
```text Sandbox
https://api.sandbox.payrequest.io/v1
```

```text Production
https://api.payrequest.io/v1
```
</CodeGroup>

## Authentication

The PayRequest API uses API keys for authentication. You can find your API keys in your dashboard under **Settings > API Keys**.

<Warning>
  Always use your test API key in sandbox mode and your live API key only in production. Never expose API keys in client-side code.
</Warning>

Include your API key in the request header:

```http
Authorization: Bearer your_api_key_here
Content-Type: application/json
```

## Quick Example

Here's a simple example to create a payment request:

<CodeGroup>
```bash cURL
curl -X POST https://api.payrequest.io/v1/payments \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 2500,
    "currency": "USD",
    "description": "Product Purchase",
    "recipient": {
      "email": "customer@example.com",
      "name": "John Doe"
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.payrequest.io/v1/payments', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 2500,
    currency: 'USD',
    description: 'Product Purchase',
    recipient: {
      email: 'customer@example.com',
      name: 'John Doe'
    }
  })
});

const payment = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.payrequest.io/v1/payments',
    headers={
        'Authorization': 'Bearer your_api_key',
        'Content-Type': 'application/json'
    },
    json={
        'amount': 2500,
        'currency': 'USD',
        'description': 'Product Purchase',
        'recipient': {
            'email': 'customer@example.com',
            'name': 'John Doe'
        }
    }
)

payment = response.json()
```
</CodeGroup>

## Response Format

All API responses are returned in JSON format. Successful requests return a `200` status code along with the requested data. Error responses include an error code and message.

### Success Response
```json
{
  "id": "pay_1234567890",
  "amount": 2500,
  "currency": "USD",
  "status": "pending",
  "payment_url": "https://payrequest.io/p/pay_1234567890",
  "created_at": "2024-01-15T10:30:00Z"
}
```

### Error Response
```json
{
  "error": {
    "code": "invalid_request",
    "message": "The amount field is required"
  }
}
```

## Rate Limits

The PayRequest API has the following rate limits:

- **100 requests per minute** for most endpoints
- **500 requests per minute** for webhook endpoints
- **1000 requests per minute** for read-only endpoints

Rate limit headers are included in every response:

```http
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 99
X-RateLimit-Reset: 1640995200
```
